GSE161895 visual package

This package builds a final figure pack for the GSE161895 branch.

Core plots
1. Master reversion index
2. Formal trajectory visuals
3. Treatment program shift comparison, whole D2_1 and Patient3 sensitivity
4. GO mechanism panels for the R-bridge
5. Variance scaling, state-age proxy, and plasticity/reversion panels

Use
Put this folder into the repo root as:
gse161895_visual_package/

Then run:
bash gse161895_visual_package/scripts/run_gse161895_visual_package.sh
